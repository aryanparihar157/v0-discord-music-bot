================================================================================
                        WHY THIS WORKS
================================================================================

Your bot uses Discord's Interactions API (webhooks), which is:
  ✓ Serverless - Works perfectly on Vercel
  ✓ Stateless - No database needed
  ✓ Fast - Responds in < 500ms
  ✓ Reliable - Discord handles retry logic
  ✓ Free - No costs on Vercel free tier
  ✓ Always-on - 24/7 without keeping a server running

Unlike traditional Discord.js bots that need persistent connections,
your bot receives webhooks from Discord and responds via HTTP.

================================================================================
                          IF SOMETHING FAILS
================================================================================

"Bot doesn't show commands"
  → Wait 30 seconds and type "/" again
  → Check Interactions Endpoint was saved in Discord
  → Verify DISCORD_PUBLIC_KEY is in Vercel env vars

"401 Unauthorized errors"
  → PUBLIC_KEY might be wrong
  → Copy it again from Discord Developer Portal
  → Re-add to Vercel env vars

"Endpoint verification failed"
  → Make sure https://your-vercel-url.vercel.app/api/discord/interactions is exactly correct
  → Check that deployment is complete and live
  → Wait 30 seconds and try again

Check Vercel logs: Vercel Dashboard → Deployments → Bot deployment → Logs
