QUICK SETUP - 5 MINUTES
================================================================================

STEP 1: Get Your Public Key (30 seconds)
────────────────────────────────────────
1. Visit: https://discord.com/developers/applications
2. Select your application
3. Go to "General Information" tab
4. Copy "PUBLIC KEY"
5. Add to Vercel:
   - Go to project settings → Environment Variables
   - Add: DISCORD_PUBLIC_KEY = (your public key)

STEP 2: Deploy to Vercel (2 minutes)
────────────────────────────────────
Your code is ready. Just:
1. Connect your GitHub repo to Vercel
2. Add environment variables (already set up)
3. Click "Deploy"
4. Wait for completion

Your bot will run at: https://[project].vercel.app

STEP 3: Set Interactions URL (1 minute)
───────────────────────────────────────
1. Go back to Discord Developer Portal
2. General Information tab
3. "Interactions Endpoint URL" field
4. Paste: https://[your-vercel-url].vercel.app/api/discord/interactions
5. Save and verify (green checkmark)

STEP 4: Invite Bot (1 minute)
──────────────────────────────
Visit: https://[your-vercel-url].vercel.app/setup
Click the invite button and select your server!

================================================================================
HOW IT WORKS
================================================================================

User types /play command in Discord
         ↓
Discord sends webhook to your Vercel bot
         ↓
Bot processes command (/api/discord/interactions)
         ↓
Bot searches for music (YouTube, Spotify, etc.)
         ↓
Bot responds to Discord
         ↓
User sees response instantly

All happens in <500ms! ⚡

SUPPORT & TROUBLESHOOTING
================================================================================

Commands not showing?
  → Wait 5-10 minutes for Discord to sync
  → Remove and re-invite the bot
  → Check bot has "applications.commands" scope

Bot not responding?
  → Check Vercel logs for errors
  → Verify all 3 environment variables are set
  → Ensure Interactions Endpoint URL is verified

Endpoint verification failed?
  → Verify URL: https://your-domain.vercel.app/api/discord/interactions
  → Check environment variables
  → Check Vercel deployment is complete
